Who is this person

My handle is Window Dump (check your man pages to learn more about me, I just output the content of a window to a file in text format). I like computers. I figured out how to program them and get them to do increasingly silly stuff - this blog and its content is just the latest example.

If you’re looking for my old fork of KeyZen, a tool used to practice keyboard layouts (i.e., Dvorak, Colemak, Hands Down), you can find it right here, in the usual location. Features sounds from some rectangle. I use Cornes btw.

Current fixations include (but are not limited to):

  • indie TTRPG’s: OSR/NSR, story games, one-page games, games that dislike categorization, etc.
  • Typst, typography, graphic design
  • the latest WoW expansion

(What does OSR stand for? Last time I checked, I got “Original Sicilian Ravioli” and that sounds delicious, so let’s go with that.)

I am currently interested in running FIST, Breakers, and Blades in the Dark. Other games I want to check out include NOVA, Lancer, Whitehack, Cairn, Troika!, and Electric Bastionland or another Into the Odd based game.

My home table is currently playing Cyberpunk RED. I started playing with them before I looked up the PbtA tag on Itch.
