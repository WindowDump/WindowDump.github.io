FIST Operative Handbook

A zine-style half-letter character sheet and rules summary for FIST, a paranormal mercenary TTRPG.

I’m a big fan of FIST, a TTRPG about paranormal mercenaries (think Metal Gear) doing jobs for regular people no one else can (or no one else wants to) during the Cold War (think The A-Team). One of my favorite parts is how quickly a character can be generated, and how flavorful these characters are. Check out FISTREF’s Random Merc page if you want to see it in action for yourself. FIST’s anti-canon sandbox defines the world with the help of the characters.

FIST’s core is short and concise, and I decided to combine a rules reference and a character sheet together for a zine-style booklet. This is particularly useful for new players who haven’t read the player rules yet. You can read both sides, follow the character creation rules on the outside, then fold it in half to maximize table space.

This was a first project in Typst so there’s probably a lot of ugly hacks in there - I might clean it up later. Possible project for the reader: replace Orbitron and Michroma with Microgramma (aka the Drake & Josh font), update letter-spacing as needed.

I also made a version that’s just the character sheet twice, so you can print out two character sheets at once!

This is kind of a cool format for a one-page game that was surprisingly easy to whip up.
