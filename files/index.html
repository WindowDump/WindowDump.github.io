Stuff I made

Here’s some game stuff I made. Source is provided so you can tweak or hack them to your liking. Change fonts, change colors, tweak the words, build a game on top of it. Or print them out and start playing.

FIST

FIST - Freelance Infantry Strike Team, a paranormal mercenary TTRPG.

Operative Handbook

Half-letter character sheet, plus character creation rules and a rules reference. Great for new players - read the whole thing, then make a character and fold it in half to maximize table space.

  • FIST Operative Handbook - PDF
  • Source Code (Typst)

Half-letter character sheet

Just the sheet from the above handbook, two per page. Print out a bunch so you’re ready to emergency insert as many mercs as you need.

  • FIST half-letter character sheet - PDF

Remix

AKA World of Dungeons (WoDu) Remix. A flexible ruleset built on World of Dungeons.

Half-letter character sheet

Inspired by the original WoDu sheet, updated with the skills from Remix. Intended to be remixed - you can easily hack it to change the background texture, header, font fill, etc.

  • Black & White - PDF
  • Low Ink, Single Contrast - PDF
  • flare, color - PDF
